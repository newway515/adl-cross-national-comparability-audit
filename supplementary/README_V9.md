## 关键术语速查

| 术语 | 含义 |
|---|---|
| ADL | Activities of Daily Living（日常生活活动）：穿衣/洗澡/进食/上下床/如厕 |
| Gateway | Gateway to Global Aging Data——将 6 个国际老龄队列的变量映射为统一 Harmonized 格式 |
| Harmonized | Gateway 提供的协调变量体系，"任何困难 = 1"的二值 ADL 编码 |
| DIF | Differential Item Functioning（差异项目功能）——同样失能水平的老人，因国家/文化不同而对同一问题给出不同回答的系统倾向 |
| IRT / 2PL | Item Response Theory / 2-Parameter Logistic Model——将失能建模为潜特质（不可直接观测的连续变量），条目参数（区分度 a、难度 b）描述条目特性 |
| E1 | 条目定义效应（条目集和阈值不同导致的患病率差异）——V9.0 确认为 ≈ 0（Gateway 已消除） |
| E2 | 评分效应（评分规则不同导致的患病率差异）——仅 CHARLS 有非零信息（原始 4 级量表） |
| E3 | DIF 效应（回答倾向不同导致的患病率差异）——非点识别，以包络区间报告 |
| E4 | 总测量位移 = E1 + E2 + E3 |
| 识别区间 (identified set) | 在无额外假设下，E3/DIF 的真实量级只能被界定在 [min, max] 范围内——跨度反映了结论对识别约束选择的敏感度 |
| 识别边界图 (identifiability frontier map) | 可视化哪些国家对的排序在什么假设下稳定、在什么假设下翻转 |
| M0–M3 | 四层分析框架：M0 原生 → M1 条目协调 → M2 评分协调 → M3 IRT 潜变量 |
| CHARLS 4 级 | CHARLS 原始数据保留了 ADL 的 4 级严重度编码（无困难/有困难能完成/需帮助/不能做），是唯一有评分梯度的数据源 |
| LD | Local Dependence（局部依赖）——条目的残差在控制潜特质后仍显著相关（如穿衣+洗澡），违反 2PL 的局部独立假设 |
| C8 | 用合成数据（注入已知真值）对分解算法做前置体检的模拟研究，已验证 2PL 估计算法在理想条件下无偏、CI 覆盖名义 |

## 从 V8.1 到 V9.0 的核心变更

| V8.1（原方案） | V9.0（重构后） | 驱动变更的评审共识 |
|---|---|---|
| 三层分解（条目/评分/DIF）为核心卖点 | 协调数据可比性审计 + 识别边界测绘为核心卖点；三层分解降为分析工具 | 四份评审一致：E1≈0 使三层叙事空心化 |
| E1≈0 藏在方法脚注里 | E1≈0 前置为"Gateway 协调质量的正面证据" | 两位同行评审独立指出应在标题/摘要显式化 |
| H2："条目定义效应 ≥ DIF 效应"（确证假设） | H2 撤销，转为描述性发现（"在协调数据上 E1 已被上游消除"） | 四份评审一致：H2 在 E1≈0 下被自身数据证否 |
| R_g ≥ 1 硬阈值判定 | 识别区间 + 排序方向不可识别判定（不硬分"可/不可"） | 四份评审一致：R_g 量纲混合+阈值无标定+部分识别下大面积失效 |
| 局部依赖（LD）为可能的 limitation | LD 诊断入主分析（Q3 残差相关 + testlet 敏感性） | 两份同行评审独立指出 C8 S11 暴露 LD 致覆盖率崩溃 |
| 选择性生存未提及 | 讨论中专段处理（定性敏感性 + 预期寿命散点图） | 单作者同行评审独立识别 |
| 期间效应/COVID 未提及 | 讨论中专段处理（SHARE W8 晚期月对比） | 两位同行评审独立提及 |
| 稀疏条目未处理 | 条条分辨率 + 留一条目敏感性入主分析 | 两位同行评审独立指出进食阳性率 < 5% |
| M2 操作化悬而未决 | 主分析 = 二值化阈值变更（"需要帮助或不能 = 1"）；严重度加权为敏感性附录 | 研究者自行决策（在四份评审意见基础上的综合） |
