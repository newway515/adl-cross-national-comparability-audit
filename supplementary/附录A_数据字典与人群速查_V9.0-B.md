## A.1 五条目 ADL 编码

| 条目 | 英文 | 变量名（协调） | 结局编码 |
|---|---|---|---|
| 穿衣 | dressing | dressa | 有任何困难 = 1，否则 0 |
| 洗澡 | bathing | batha | 有任何困难 = 1，否则 0 |
| 进食 | eating | eata | 有任何困难 = 1，否则 0 |
| 上下床/起身 | getting in/out of bed | beda | 有任何困难 = 1，否则 0 |
| 如厕 | toileting | toilta | 有任何困难 = 1，否则 0 |

主结局：5 条目任一为 1 → "任一 ADL 困难 = 1"。

**CHARLS 原始 4 级量表（唯一非二值起点）：** 原始变量 db010–db014，编码 1=无困难 / 2=有困难仍能做 / 3=需帮助 / 4=不能做。协调默认将 2/3/4 映射为"任何困难=1"。评分层敏感性中，主口径 M2b 改用"3 或 4 = 1"阈值。原始缺失码（如拒答/不知道）须在抽取 CHARLS 原始表时按其 codebook 核验后再折叠。
