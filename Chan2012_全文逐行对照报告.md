## 二、七维度定量/定性对照——核心问题：Chan 2012 是否"已经做了本课题要做的事"？

### 维度 1：DIF 检测方法的深度与透明度

| 方面 | Chan 2012 | 本研究 (V9.1) |
|---|---|---|
| **似然比检验** | ✅——使用 IRTLRDIF 做 χ² 差异检验 (嵌套模型比较) | ❌——不使用假设检验框架，改用包络区间 |
| **效应量 (DIF magnitude)** | ✅——报告概率差异的具体数字 (如"最大差异 40%") + TCC 期望得分差异 | ✅——以患病率量纲 (pp) 报告 E3/DIF 的识别区间 |
| **多重比较校正** | ✅——Benjamini-Hochberg 校正 | ❌——因不使用假设检验，多重比较以区间估计主分析 + BH 校正为副 (V9.1 §9.5) |
| **识别不确定性** | ❌——未讨论——单一参照 (HRS) + 迭代锚条目，未检验约束敏感度 | ✅——四种识别结构包络 + 随机锚定敏感性 + 似然面扫描——核心方法学贡献 |
| **非参数验证** | ❌——仅使用参数化 IRT-DIF | ✅——Mantel-Haenszel + 逻辑回归 DIF 独立三角验证 |
| **模型拟合诊断** | ❌——未报告 (Multilog 黑箱) | ✅——收敛诊断 / 起始值敏感性 / GH 节点数敏感性 / S10 (2PL 误设) 模拟体检 |
| **设计方差** | ❌——未处理 (推测 iid 标准误) | ✅——≥1000 完整管线 bootstrap 传播复杂调查设计方差 |

**结论：** Chan 2012 的 DIF 检测方法在 **2002-2004 年数据 + 含 11 个条目的条件下是合理的**——似然比检验 + 效应量 + 多重比较校正，比当时多数同类研究更规范。但**在本研究的识别约束下 (5 条二值条目、无可用 vignettes)，Chan 2012 的单一参照 + 统计检验框架不足以处理 DIF 与 impact 的混淆——这是本研究引入部分识别的主要动机。** Chan 2012 报告了 DIF 在条目水平和标尺水平的量级，但**未讨论不可识别性本身**，也未检验约束选择的敏感度。

### 维度 2：Chan 2012 是否量化了"DIF 贡献占总跨国患病率差异的百分比"？

**答案：否。**

Chan 2012 做了以下与 DIF 量化相关的事：
1. 报告了 **Test Characteristic Curve (TCC) 期望得分差异**——HRS vs SHARE 最大 0.36 (0–11 标尺)，HRS vs ELSA 范围 −1.55 至 0.53。
2. 比较了**原始 sum score 和 DIF-adjusted IRT score 的均值**——HRS 1.09→1.70, ELSA 0.81→1.46, SHARE 0.68→1.17。
3. 比较了 DIF-adjusted 和 unadjusted IRT scores 在不同**人口学亚组**中的均值差异显著性变化。

**但：**
- TCC 差异是在"0–11 标尺"上报告的，**不是在"患病率量纲"上报告的**。0.36 分在 0–11 标尺上是多少患病率百分点？Chan 2012 没有转换。
- 原始 score 到 IRT score 的均值变化 (+0.56-0.65 分) 大部分来自**IRT 本身的distributional effect**（IRT 将得分分布拉升），而非 DIF adjustment per se——作者自己在第 130 页承认了这一点："Using a score generated by IRT methods has a stronger effect on scores at the lower end of the scale, thus potentially affecting more people than DIF adjustment."
- **从未计算过相当于 E3（DIF 对患病率差异的贡献）的量。** 本研究则把 DIF 的效应量以患病率量纲 (percentage points) 显式表达。
- **从未计算过相当于 E1（条目定义效应）或 E2（评分效应）的量。** Chan 2012 直接将三个调查的条目视为已协调 ("any difficulty" coding)，没有审计协调流程本身的完整性。

**这一确认是至关重要的——这意味着本研究相对于 Chan 2012 在"DIF 贡献的患病率量纲量化"这一具体指标上具有净增量。**

### 维度 3：跨调查数量与地理覆盖

| 方面 | Chan 2012 | 本研究 (V9.1) |
|---|---|---|
| **调查数** | 3 | 6 |
| **国家数** | 3 (美国+英格兰+欧洲12国pooled) | 13 (中国+英格兰+美国+印度+墨西哥+德/捷/爱/斯/波/西/意/以) |
| **地理覆盖** | 北美 + 西欧 | **北美 + 西欧 + 东欧 + 东亚 + 南亚 + 拉美 + 中东** |

**结论：** Chan 2012 覆盖了 HRS 家族最早的三位成员。本研究新增了 CHARLS、LASI、MHAS——三个来自中低收入国家的队列——这是重要的地理扩展，也是 DIF 可能最大的"跨文化带"。

### 维度 4：条目/评分层审计

| 方面 | Chan 2012 | 本研究 (V9.1) |
|---|---|---|
| **条目定义层审计 (E1)** | ❌——未涉及。直接将三个调查的条目视为可比较，未检验原始条目集/题干措辞的差异。 | ✅——C3 审计 + M1 抽取证实 E1≈0。**"E1≈0"本身是对 Gateway 协调完成的正面认证——这是一个独立的发现。** |
| **评分规则层审计 (E2)** | ❌——未涉及。三调查均用"任何困难"二值，未检验评分规则的差异。 | ✅——CHARLS 原始 4 级量表→M2b 改用更高的失能阈值 (需帮助或不能=1)，CHARLS 成为 E2 唯一非零的案例。 |
| **三来源框架** | ❌——未将条目/评分/DIF 放在同一量纲下同时呈现 | ✅——M0→M1→M2→M3 四层框架——即使 E1/E2 为空，框架本身是一个新的分析视角 |

**结论：** Chan 2012 没有审计过 Gateway 协调本身的完整性。本研究的 E1≈0 发现是成百上千协调数据使用者需要的正面认证。**"协调之后还剩多少不可比"这一审计性问题——包括其非 DIF 层的检查——是本研究独有的。**

| 方面 | Chan 2012 | 本研究 (V9.1) |

## 三、核心发现总结

### 3.1 Chan 2012 没有做的事情（= 本研究的真实净增量）

1. **没有在患病率量纲上量化 DIF 贡献。** TCC 差异是期望得分差异 (0–11 scale)，不是患病率百分点差异 (percentage points)。本研究将 DIF 效应显式表达为 E3(pp) = P(M2) − P(M3)——同尺度、同量纲。

2. **没有做条目定义层和评分规则层的显式审计。** Chan 2012 直接把三调查的 ADL/IADL 当作"同一套条目"来做 DIF，没有验证这个前提是否成立。本研究 M1 抽取证实——前提成立 (E1≈0)，但这本身就是一个需要被审计的事实。

3. **没有将 SHARE 内部 8 国的 DIF 分离开来。** Chan 2012 将整个 SHARE (12 国) 当作单一组，无法检验翻译型 DIF vs 体系型 DIF。

4. **没有处理不可识别性问题。** 单一 HRS 参照 + 统计检验框架对"DIF 与 impact 不可同时识别"未做讨论。

5. **没有把 DIF 贡献与决策联系起来。** "建议使用 DIF-adjusted 分数"是方法学建议，不是决策框架。本研究"可比性标记"直接回答"这对国家的排序能不能用于负担排名"。

### 3.2 引言的 gap statement 应该如何写 (基于此对照)

**推荐措辞 (英文原意，正式稿件再润色)：**

> Chan et al. (2012) demonstrated that meaningful ADL/IADL-DIF exists between HRS and ELSA, but not between HRS and SHARE, using likelihood-ratio DIF detection in a 2PL IRT framework. Their analysis reported DIF magnitude on the 0–11 expected-score scale and recommended DIF-adjusted IRT scoring. **Our study extends this work in three ways. First, we audit the upstream harmonization itself** — confirming that item-definition differences (E1) have been fully eliminated in Harmonized ADL data, and that scoring-rule differences (E2) persist only for the one cohort (CHARLS) that retained a 4-level severity scale. **Second, we express DIF displacement in the same percentage-point metric as the observed cross-national prevalence differences**, decomposing the total measurement displacement into item, scoring, and DIF components — enabling direct calculation of the DIF-attributable fraction of each country-pair gap (RQ1, RQ2). **Third, we replace the point-estimate-plus-significance-test framework with a partial-identification approach**: acknowledging that DIF and true latent-mean differences are not simultaneously point-identifiable with five binary items, we report the identified set across four defensible identification structures and construct an identifiability-frontier map.

V9.1 方案 §3.2 的 Chan 2012 对照表中，"DIF 效应量化"行应更新为：

| 维度 | Chan et al. 2012 | 本研究 |
|---|---|---|
| DIF 效应量化 | **已量化**——TCC 期望得分差异 (0–11 scale): HRS-SHARE max 0.36, HRS-ELSA −1.55至0.53。原始 vs DIF-adjusted IRT 均值: HRS 1.09→1.70, ELSA 0.81→1.46, SHARE 0.68→1.17。**未以患病率量纲 (pp) 表述，未分解为 E1/E2/E3 层。** | 以患病率量纲 (pp) 量化 DIF 对跨国差异的贡献 (E3)，与 M0 患病率同尺度可比；E1/E2/E3 三层同框呈现 |

## 四、对课题的总体影响

**此对照确认了本课题的净增量在可查验范围内依然成立。** Chan 2012 做了 DIF 检测并报告了效应量——这些不构成重叠。本课题的增量在：

1. **患病率量纲的 E3 量化** (不是 TCC 期望得分差异)
2. **E1/E2 层的显式审计** (协调完整性)
3. **识别边界的部分识别表述** (不可识别性 → 特征而非 bug)
4. **决策转化** (可比性标记)
5. **地理覆盖** (新增 CHARLS/LASI/MHAS 三个中低收入国家队列)
6. **设计方差** (≥1000 bootstrap 全管线传播)
7. **非参数 DIF 独立三角验证** (Mantel-Haenszel + 逻辑回归)

**论文引言可以诚实地写：** Chan 2012 在 3 个调查 (HRS/ELSA/SHARE pooled) 上用 11 个条目 + IRT-DIF 证明了 ELSA 部分条目存在 DIF，并报告了期望得分量纲 (0-11 scale) 的 DIF 量级。我们的工作在此基础上：(a) 把 DIF 贡献以患病率百分点量纲显式表述，使分解与原生患病率差异同尺度；(b) 审计了 Gateway 协调本身的完整性；(c) 用部分识别框架替代了单一参照假设；(d) 将其做成可以直接告诉 GBD 使用者"哪些对的比较可信"的决策工具。
